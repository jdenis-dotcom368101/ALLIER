Fonctionnement

Cet outil analyse les Règlements de Consultation (RC) au format PDF grâce à l'IA Claude (Anthropic). En un clic, il génère automatiquement :

  • Le formulaire DC1 pré-rempli (sections A et B depuis le RC, section D avec les données LACLIM)
  • Le formulaire DC2 pré-rempli (mêmes règles que DC1)
  • L'Excel du registre AO mis à jour et cumulatif

Clé API Anthropic

Une clé API Anthropic est nécessaire. Elle est stockée uniquement dans votre navigateur (localStorage) et n'est transmise qu'à l'API Anthropic lors des analyses.

Pour en obtenir une (gratuit pour tester) : console.anthropic.com
